* Repository for Docker image of the spine classifier
** Dependencies:
- docker
- numpy
- opencv
- No GPU!
** Build:

#+BEGIN_SRC bash
sh build.sh
#+END_SRC

** Run
#+BEGIN_SRC 
sh run.sh [path to the data folder] [data file]
#+END_SRC

ex: sh run.sh /home/deepinfer/image_data img1.jpg
